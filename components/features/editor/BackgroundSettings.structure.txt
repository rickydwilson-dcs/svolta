State Management:
-----------------
Reads:
  - backgroundSettings (from useEditorStore)
  - isPro (from useUserStore)

Writes:
  - setBackgroundSettings (to useEditorStore)

Internal State:
  - customColor: string (for color picker)

Event Handlers:
---------------
1. handleTypeChange(value: string)
   - Changes background type (original/transparent/solid/gradient)
   - Shows upgrade prompt for free users
   - Sets default values for solid/gradient types

2. handleColorSelect(color: string)
   - Updates solid background color
   - Shows upgrade prompt for free users

3. handleGradientSelect(presetId: string)
   - Updates gradient background
   - Shows upgrade prompt for free users

4. handleCustomColorChange(e: ChangeEvent<HTMLInputElement>)
   - Updates custom color from HTML5 color picker

- Gradient previews rendered directly in CSS
- Color swatches with aspect-square ratio
